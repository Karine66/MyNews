# MyNews
My news est une application basées sur les news du New York Times et qui permet d'accéder aux articles publiés par ce journal

**Compétences utilisées**
- Butterknife
- RecyclerView
- Material Design
- Retrofit
- RX Java
- Glide
- New York Times API
- WebView
- Shared Preferences
- Notifications
- Espresso (Test instrumnentés)
- Tests Unitaires
- BitRise (Test d'intégration)
